表格

​ · < table >、< tr >、< td >、< th >（居中加粗）、< thead >、< tbody >、< tfoot >

​ · 加入align=”center”可使表格内容居中

​ · 单元格合并：找到需要合并的单元格并删去一定数量（row合并留上面的，col无要求），在剩余单元格上添加属性rowspan/colspan=“n（需要合并的数量）”

背景

​ bgcolor：背景颜色

​ background：背景图片（重叠的）

图片

​ 图片是浮在网页上的，而背景图片是贴在网页上的。

​ title=“显示输入鼠标悬停时想要显示的内容”

​ alt=“显示图片加载不出来时的内容”

超链接

​ target=“_blank”：新开页面

​ _blank/ _self/ _top/ _parent

列表

​ 1.无序列表

​ < ul >下辖< li >，type=“选择你想要的样式”

​ 2.有序列表

​ < ol >下辖< li >

​ 3. list-style：none 取消列表自带表头

表单

功能约等于可以收集数据的超链接

​ 1. action=“（输入提交的url）”

​ 2. < input type=”submit” > 表示这是一个具有提交表单功能的按钮

​ 其中，name=“”是必要的，value=“”对于输入的内容可以不需要（因为用户输入的内容就是实时更新的value值）。提交后在网页上显示为形如“?username=jack22&userpwd=213”的字样

​ 3.各种控件：

​ type=“file”（input型）

​ type=“textarea”（input型）

​ type=“radio” （input型） checked表示已选中

​ maxlength=“n” 最大输入长度

​ select下辖option表示下拉列表 selected表示已选中 multiple=“multiple”表示多选

​ 4. method

​ 默认为get不加密

​ post为加密

​ 5. hidden

​ 网页看不到，但提交时数据也提交给服务器

​ 6. readonly和disabled

​ 相同点：都是只读

​ 不同点：readonly可以提交，disabled不能提交

id属性

​ id是身份证号，有唯一性

div和span

​ < div > 标签定义 HTML 文档中的一个分隔区块或者一个区域部分，常用于组合块级元素，以便通过 CSS 来对这些元素进行格式化

​ < span > 标签用于对文档中的行内元素进行组合

​ div独占一行，span不会

常用属性

​ placeholder：浮于输入框上方的虚拟文字

​ autocomplete：自动提示填充（=“on”）

​ autofocus：自动获得焦点

​ datalist：下拉列表

​ enctype ：规定在发送到服务器之前应该如何对表单数据进行编码。

​ label：为 input 元素定义标注，< label > 标签的 for 属性应当与相关元素的 id 属性相同。

​ list-datalist-option: input=”text”里的一种类型，网页中显示为下拉选项。list的值和datalist的value需要相等。

html5新特性

​ required：必填项

​ pattern：正则

​ novalidate/formnovalidate：设置了必填但表单提交不验证

​ 约束验证api：

​ validity

CSS

​ 使用格式： 选择器{内容1；内容2} 注意：这里没有”.”

· 常用标签（查文档）

​ font-size字号大小；font-family字体；font-weight字体加粗； font-style字体风格（简写顺序：font-style font-weight font-size（不能省） font-family（不能省））

​ 颜色尽量使用十六进制，可简写；rgb(a)形式 #3c3c3c

​ line-height行高（trick：在一行的盒子内，设定行高=盒子高度可使文字垂直居中）；text-align水平对齐方式；text-indent首行缩进(em为单位，1em为一个汉字单位)

​ letter-spacing字间距；word-spacing单词间距（只针对英文）

颜色半透明CSS3

​ rgba（z,x,c,v），v范围0~1，表示透明度，越靠近0越透明

文字阴影CSS3

​ text-shadow：水平位置（阴影左右方向动），垂直位置（阴影上下方向动），模糊距离，阴影颜色

​ 可以自己到网页上F12调整完毕后记录数据再回填到css中

· 选择器

类选择器

​ style中：.命名{内容1；内容2}

​ body中的被选元素：class=“命名”

​ “其中，命名不要用下划线而是短横线，不能用纯数字； 多个类名之间用空格分隔 “

id选择器

​ style中： #命名{内容1； 内容2}

​ body中的被选元素： id=“命名”

通配符选择器

​ sytle中： *{内容1； 内容2}

​ 作用是选中所有标签

伪类选择器

​ · 链接伪类选择器（主要针对a），尽量按照lvha的顺序 以上“a”可全用class类名代替

​ a:link{} 未选中时的状态

​ a:visited{} 访问过后的状态

​ a:hover{} 鼠标悬停时的状态

​ a:active{} 鼠标点击时的状态

​ · focus伪类选择器——把有光标的内容选中显示，主要针对input类型

​ ：focus

​ · 结构伪类选择器

​ :first-child{} 第一个内容

​ :last-child{} 最后一个内容

​ :nth-child(选择第几个){} 自主选择内容

​ 其中，（even)选择出偶数行/个内容，(odd)选择奇数行/个内容；（）内也可以写公式，例如2n或2n+1等等

​ :nth-last-child{} 倒数选择

​ · 目标伪类选择器

​ :target{} 谁被选中谁变化

属性选择器

​ style内：选择器[内容]，body中写入相应内容

伪元素选择器

​ 选择单个元素，双冒号：：

​ ::first-letter 第一个字

​ ::first-line 第一行

​ ::selection 选中时状态改变

​ ::before 放在盒子内容的“里”前面

​ ::after 放在盒子内容的“里”后面
